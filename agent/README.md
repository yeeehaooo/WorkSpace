- The governance core (Source of Truth) for AI behaviors and engineering policies.
- Tool-agnostic rules: architecture principles, review policy, naming, exception policy, security guidance.

## What this folder must NOT contain
- Tool-specific instructions (Cursor/Claude/VScode). Put those under /adapters or tool folders.
- Project-specific code or templates tied to a single repository.
- Large implementation snippets that belong to skill packs (put under /skills).
